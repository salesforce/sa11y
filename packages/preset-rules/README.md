## Usage

```javascript
import axe from 'axe-core';
import { extended } from '@sa11y/preset-rules';

const a11yResults = await axe.run(extended);
console.log(a11yResults);
```

## Rulesets

sa11y provides two pre-configured rulesets curated from [axe's available rules](https://github.com/dequelabs/axe-core/blob/develop/doc/rule-descriptions.md): Recommended and Extended.
